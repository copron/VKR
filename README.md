# Оценка качества проведения A/B‑теста

Этот репозиторий содержит:
- ноутбук с различными подходами к системе, обучением и анализом ML-моделей;
- Streamlit‑приложение для автоматической оценки качества оформления карточки A/B‑теста по загруженному .docx‑файлу.

Все исходные данные были **анонимизированы** в соответствии с NDA компании: любые чувствительные поля и идентификаторы заменены или удалены. Код демонстрирует основную логику и этапы работы, не раскрывая коммерчески чувствительную информацию.

## Установка зависимостей

Рекомендуется использовать отдельное виртуальное окружение.

```bash
pip install -r requirements.txt
```

## Запуск Streamlit‑приложения

Из корневой папки проекта:

```bash
streamlit run main.py
```

После запуска откройте указанный в терминале локальный URL, загрузите .docx с карточкой теста и получите автоматические оценки по этапам, критериям и всему тесту в общем.
